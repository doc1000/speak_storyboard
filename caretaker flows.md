You already have a very strong seed set because they’re:

* emotionally real
* operationally common
* visually representable
* cognitively important

What you want now is **coverage across caregiver domains** so the audience sees this is bigger than a gimmick.

The trick:

* avoid “everything”
* focus on **high-frequency caregiver communication friction**

# Core Caregiver Categories

You should probably organize the demo around these:

| Category                    | Why it matters             |
| --------------------------- | -------------------------- |
| Food & hydration            | Daily repetition           |
| Medication                  | Safety-critical            |
| Movement & navigation       | High confusion/frustration |
| Visitors & schedule changes | Anxiety reduction          |
| Financial/simple tasks      | Cognitive sequencing       |
| Transportation              | Independence tension       |
| Hygiene & health            | Sensitive but essential    |
| Emotional reassurance       | Huge emotional impact      |
