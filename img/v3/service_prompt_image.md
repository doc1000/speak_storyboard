Create a single warm, emotionally clear caregiver communication illustration for an elderly adult.

Primary goal:
The image must communicate one simple message quickly and clearly through people, objects, and gesture.

Visual style:
- warm polished 3D illustration or soft high-quality digital illustration
- gentle healthcare-education look
- friendly expressive human faces
- emotionally readable body language
- clean, simple composition
- soft natural indoor lighting
- elderly grandmother with gray hair
- caring middle-aged caregiver
- clear focal subject
- minimal background clutter
- large recognizable objects
- realistic proportions
- calm reassuring mood

Hard constraints:
- show full faces when faces are visible
- do not crop, hide, blur, block, or obscure faces
- no comic strip layout
- no multiple panels unless explicitly requested
- no text, labels, speech bubbles, captions, or handwriting in the image
- no poster design
- no sketch style
- no black-and-white line art
- no harsh contrast
- no fragmented composition
- no extra objects that do not help the message
- no photorealism

Communication rules:
- communicate one idea per image
- make the action and emotional intent obvious at first glance
- prefer familiar home-care scenes
- use clear pointing, offering, sitting, walking, or returning gestures
- if a question cue is needed, use a simple large question mark in the top right corner only if it improves clarity
- if a movement cue is needed, use one bold simple arrow only if it improves clarity
- if a time cue is needed, use one simple clock only if it improves clarity

Priority:
clarity over style
emotion over decoration
familiarity over novelty
